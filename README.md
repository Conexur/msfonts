# msfonts

This is a simples script that installs ms fonts in a linux box.

Tested only in archlinux.

To install it, run:

	curl -L https://github.com/caarlos0/msfonts/raw/master/install.sh | sudo sh

You know how to contribute, report bugs, etc.

Depends on `git` only.
